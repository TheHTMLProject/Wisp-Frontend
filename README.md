Setup

1. Install dependencies:

```bash
pnpm install
```

2. (Optional) Create `.env` in project root:

```env
DISCORD_WEBHOOK_URL={url here}
```

3. Start the server:

```bash
pnpm start
```

4. Open:

`http://localhost:9999`

Project Branding

This project is branded as `The HTML Project Connect` (`Connect` for short).

The server also honors `PORT`, for example:

```bash
PORT=9999 pnpm start
```
